Visualising Irelands Data

Welcome this web site allows you to visualize all of the Central Statistics office Irelands data.

Begin Visualizing

This Application groups the data sets from the Central Statistics Office by Geographical Region, Unit Base,Time Series, and Feature.

  1. By Region click on the Maps button on the toolbar to begian viwing the data by region.
  2. This will display the data by geographical region in the side bar.
  3. Click on a region and the data sets from that region will be displayed.
  4. Then select a data set and a map of Ireland will be displayed.
  5. If there is more that one regional division in the set both will be displayedYou may have to scroll to both maps

  1. TheTime is used the same click on the menus to expand then click on a data set name to display a graph.
  2. Time Displays data in a time series format the features are grouped by unit each feature in a data set is given a different coloured line
  3. All the data sets in each section can be easly compaired and ploted on the same graph as they have the same unit. This is a useful feature and will be expended on soon.

  1. Feature(temperaly out of action) groups the datasets which share similar variables such as Time in Years or County
  2. Feature is used in the same way as the others
