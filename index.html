Quick Start

This Application visualizes the data from the Central Statistics Office by Geographical Region, and in Time Series by Unit.


  • By Region click on the Maps button on the toolbar to begin viewing the data by region.
  • This will display the data by geographical region in the side bar.
  • Use the tree based menu to select a Statistic you would like to map.
  • If there is more that one regional division in the set both will be displayed You will have to scroll to view both maps

  • TheTime feature draws a time series line chart with each variable displaying a different colour line
  • In the menu the Statistics are grouped by unit
  • Once a line chart is displayed you can click on the coloured squares on the top to remove a line. The box will become grayed out. The line can be added back to the chart by clicking on the box again
